# Image Editor

University project 2025

## Run

```bash
docker compose up
```

For cpp hotreloading:

```bash
python watch_cpp_folder.py
```

## Technologies

- React
- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI
- WASM (WebAssembly)
- Emscripten
- C++
- Docker
